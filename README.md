🧴 Skin Care Sales Analysis Dashboard

This project analyzes a real-world skin care sales dataset sourced from Kaggle (2,00,000+ rows).

🔹 Project Overview

The dataset was cleaned and transformed using Microsoft Excel. Additional calculated columns were created to enhance business insights. The cleaned data was then connected via SQL Server to Power BI for visualization and dashboard development.

🔹 Tools Used

MS Excel (Data Cleaning & Transformation)

SQL Server (Database Integration)

Power BI (Dashboard & Visualization)

🔹 Key Insights

Sales performance analysis by product category

Revenue trend analysis

Customer purchase behavior patterns

Category-wise performance comparison
